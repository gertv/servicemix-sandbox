Welcome to the ServiceMix Quartz Binding Example
================================================

This example shows are basic flow of events, started by a Quartz time activating a JMS Message sender

Before starting this example, run

ant setup

To start the servicemix container, run:

../../bin/servicemix servicemix.xml

For more information on running this example please see:

http://incubator.apache.org/servicemix/Basic
