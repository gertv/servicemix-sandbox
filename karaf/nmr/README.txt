Welcome to Apache ServiceMix NMR
====================================
Apache ServiceMix NMR is a JBI container built as a set of OSGi bundles
and meant to be deployed on top of Apache ServiceMix Kernel.


Getting Started
===============
For an Apache ServiceMix NMR source distribution, please read
BUILDING.txt for instructions on building Apache ServiceMix NMR.

For an Apache ServiceMix NMR binary distribution, please read
RELEASE-NOTES.txt for the list of supported and unsupported features.
ServiceMix NMR distribution contains a set of OSGi bundles meant to
be deployed onto an OSGi Runtime, preferably ServiceMix Kernel.
To deploy the NMR on ServiceMix Kernel, simply copy the bundles inside
the lib directory into the deploy directory of ServiceMix Kernel.

Alternatively, you can also find out how to get started here:
    http://servicemix.apache.org/nmr/

If you need more help try talking to us on our mailing lists
    http://servicemix.apache.org/mailing-lists.html

If you find any issues with ServiceMix NMR, please submit reports 
with JIRA here:
    http://issues.apache.org/activemq/browse/SMX4NMR

We welcome contributions, and encourage you to get involved in the 
ServiceMix community. If you'd like to learn more about how you can 
contribute, please see:
    http://servicemix.apache.org/contributing.html

Many thanks for using Apache ServiceMix.
